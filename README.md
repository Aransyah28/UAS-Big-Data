# uas-big-data
About Aplikasi web untuk menampilkan hasil analisis Machine Learning terhadap kasus DBD di Jawa Barat. Sistem ini menggunakan data real dari tahun 2016-2024 untuk menganalisis kasus DBD. Aplikasi menampilkan visualisasi interaktif seperti scatter plot, line chart, bar chart, dan pie chart untuk memudahkan pemahaman data.
